# Tries to build and run source code


## Arguments

### build
tries to make a develop build

### run
tries to run the executable

### release
tries to make a develop release

### lint
run a linter over file/project


## Supported
- Makefile
- Cargo project
- index.*
- main.*
- test.*

### Types
.js
.rs
.cpp
.c
.lua
.bash
.sh
